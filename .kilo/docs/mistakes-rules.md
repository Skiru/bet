# Betting Mistakes & Hard Rules — Compact Reference

> Apply at COUPON CONSTRUCTION stage. Candidates still appear in matrix (R3 compliance).

## ⛔ HARD RULES (violation = reject from coupon)

### TENNIS_SETS_001 — Over 3.5 Sets
ALL conditions required: ranking ≤15 gap, competitive H2H (no 3-0), BOTH <60% straight-set L5, no surface mismatch, no injury return. ANY fails → REJECT O3.5.

### TENNIS_GAMES_001 — Underdog Games Over
Ranking gap <50, hold rate >50% L5, odds ≤2.00. Higher odds = bookmaker expects blowout.

### HANDBALL_001 — Handball ML in AKO
Implied ≥70% (odds ≤1.43). NEVER same handball pick in 2+ coupons. Prefer totals/HC.

### GOALS_001 — Over X Goals
Combined L5 goals MUST > line + 0.5 buffer. Below → FLIP to UNDER or SKIP.

### UNDER_GOALS_001 — Under 2.5 Restrictions
FORBIDDEN when: team needs to win + combined L5 > 2.5 + odds ≥2.00. Never in 2+ coupons.

### LOWER_LEAGUE_001 — Lower League ML
NEVER ML below 3rd tier in AKO. 2./3. Liga ML only at odds ≤1.40. Prefer stat markets.

### SOT_001 — Shots on Target Over
Combined L5 SOT must > line + 1.5 buffer. Below line → INSTANT REJECT. Dead rubber → subtract 1.5.

### CORNERS_CONTEXT_001 — Dead Rubber Corners
Dead rubber → subtract 2.5 from expected corners. Post-penalty < line → REJECT.

### BTTS_CONTEXT_001 — BTTS Restrictions
Relegation = default NO BTTS. Never BTTS live at 0-0 30'+ in tight match.

### CORRELATION_001 — Pick Repetition (MOST CRITICAL)
MAX 1 coupon per unique pick. NEVER repeat same bet across multiple coupons.

### SAFETY_FLOOR_001 — Minimum Safety Score
Picks below safety threshold → Extended Pool only, not core coupons.

## Catastrophic Pattern (2026-05-24)
4 losing picks × 2.25 avg coupons = 9 losses from 4 mistakes. Correlation kills bankrolls.
