=== ROOT CAUSE ===
HTTP 400 Bad Request caused by duplicate system messages in conversation array.

The soak harness line 109:
  conversation = [conversation[0], *conversation[-10:]]

This slice operation includes conversation[0] (the system message) AND then
includes conversation[-10:] which also includes the system message when the
conversation has fewer than 11 messages.

- after append: [system_msg, user_msg]

=== FIX ===
Change line 109 from:
  conversation = [conversation[0], *conversation[-10:]]

To:
  conversation = [conversation[0], *conversation[-10:]][1:]  # Skip first in slice

Or better, preserve system and take last 10 non-system messages:
  conversation = [conversation[0]] + [m for m in conversation[-10:] if m != conversation[0]]

Or simplest fix - just take last 10 messages without duplicating:
  conversation = conversation[-11:]  # Keep system + last 10

Actually, the intent is: keep system message + last 10 messages.
The bug is that conversation[-10:] includes the system message when len < 11.

Correct fix:
  conversation = [conversation[0]] + conversation[-10:]
  
But this still duplicates when len(conversation) <= 10.

Best fix - ensure no duplicate system:
  conversation = [conversation[0]] + [m for m in conversation[1:]][-10:]

This keeps the system message at position 0, then the last 10 non-system messages.
